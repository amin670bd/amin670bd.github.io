System Abstract & Objective

Designed and developed a console-based high-precision automation suite for industrial aluminum production. This system acts as a real-time data bridge between physical Yaohua XK3190-D# Indicators and LightHouse ERP.

The solution replaces manual weight entry with an sub-second automated polling mechanism that identifies stable weights and validates them against a database of 1,500+ master die charts, triggering label printing in milliseconds.

Precision Engineering & Program Flow

  • System Initialization:

    The engine dynamically parses WEIGHT_CHART_MASTER.csv into a high-speed memory dictionary for O(1) instant lookup of 1,500+ Die profiles including Thickness, TargetAvg, and weight ranges.

  • Industrial Scale Bridge:

    Implements a 200ms polling cycle on Scale I/O file X:\wbdata1.wbg. Filters raw RS232 noise and ensuring only "Stable" weight signals are captured.

  • Algorithm & Adjustment:

    Automatically calculates per-piece net weight. Applies dynamic ¬±ADJ (Tolerance Adjustment) logic allowing supervisors to override minor environmental variances like coating thickness.

  • ERP Bridge Automation:

    UI automation pushes verified production logs directly into LightHouse ERP files, reducing nearly 500-700 minutes of daily manual entry bottlenecks.
