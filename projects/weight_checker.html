Project Details

Designed and developed a custom desktop suite to integrate physical industrial weight indicators with LightHouse ERP. This application serves as the primary gateway for the Packing Section, ensuring that every aluminum bundle is weighed, verified, and logged automatically without manual data entry.

  • Hardware Integration: Implemented Serial Port (RS232) communication for real-time capture from Yaohua XK3190-D indicators.
  • Workflow Automation: Seamlessly transfers validated weight data directly to ERP fields.
  • Throughput Optimization: Scaled daily label printing from a manual 2,500 to an automated 8,000+ per day.
  • Dynamic QC Logic: Performs instant 10-tier weight verification against master production charts.
