Abstract & Objective

Designed and deployed an industrial desktop application suite to unify the physical production floor with digital ERP logic. The system acts as a high-speed data bridge between Yaohua XK3190-D# indicators and LightHouse ERP.


Before this solution, packing data was entered manually, leading to human errors and production bottlenecks. My solution identifies stable weights, validates them against 1,500+ aluminum die profiles, and automates label triggering in milliseconds.

Core Technical Implementation

  • Scale Integration: Raw RS232 stream processing from Yaohua XK3190-D Weight Meter.
  • Verification Engine: Real-time cross-referencing of weight strings against master CSV charts.
  • ERP Bridge: Automated data population into LightHouse ERP using C# automation tools.
  • Windows 11 Ready: Engineered to run on low-latency OS environments with 99.9% uptime.
  • Smart Throughput: Logic-driven label printing that scaled production from 2.5k to 8k daily units.
