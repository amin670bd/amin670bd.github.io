System Abstract & Objective

Designed and developed a console-based high-precision automation suite for industrial aluminum production. This system acts as a real-time data bridge between physical Yaohua XK3190-D#Indicators and LightHouse ERP.


The solution replaces manual weight entry with an sub-second automated polling mechanism that identifies stable weights and validates them against a database of 1,500+ master die charts, triggering label printing in milliseconds.

Program Flow

Precision Engineering & Program Flow

  • System Initialization: The engine dynamically parses the WEIGHT_CHART_MASTER.csv into a high-speed memory dictionary. This allows for instant O(1) lookup across 1,500+ Die profiles, mapping specific attributes: DieID ‚Üí Category | Thickness | TargetAvg | MinWeight | MaxWeight
  • Industrial Scale Bridge: Implements a 200ms polling cycle on the Scale I/O file X:\wbdata1.wbg. The software filters raw RS232 weight noise and ensures only "Stable" weight signals are captured, preventing data fluctuation during the packing process.
  • Algorithm & Adjustment Logic: Automatically calculates the per-piece net weight based on total bundle pieces. It applies a dynamic ¬±ADJ (Tolerance Adjustment) logic that allows authorized floor supervisors to override minor environmental variances (like profile coating thickness) while maintaining the 10-tier QC guardrails.
  • ERP Bridge Automation: Utilizes high-precision UI automation and weight scale app to push verified production logs directly into weight data file used byLightHouse ERP. This single-click synchronization eliminates the 5-7 minute manual data entry and checking bottleneck per bundle. reducing nearly 500-700 munites daily to production targets
